IDA TX19A plugin

What

This is a plugin for IDA Pro 6.x (MIPS support required!) that implements Toshiba's TX19A MIPS extension.

Author

JollyRogerXP

Compilation

This plugin can be compiled using Visual Studio, following the instructions in the "IDA PLUG-IN WRITING IN C/C++" document (see below).

History

The plugin was written by JollyRogerXP in 2009 while reversing Canon still and video cameras. He originally attached the source codeto a forum post on the CHDK forums, but the link to the source code is now dead.

After a bit of hide and seek, we succeeded in getting in touch with JollyRogerXP, who managed to provide us with a copy of the source code and allowed us to use it and host it on this site
